# 日経225トレーディングゲーム 🎮

日経平均株価に連動する架空の金融商品を売買できるWebゲームです。

## 特徴

- 📊 リアルタイムの日経225価格データ（約1分ごとに更新）
- 💰 仮想通貨100万円でスタート
- 📈 買い・売り取引機能
- 📜 取引履歴の表示
- 💾 進行状況の自動保存（LocalStorage使用）
- 🎨 レスポンシブデザイン
- 🚀 バックエンド不要（GitHub Pagesで公開可能）

## 使い方

1. ブラウザで `index.html` を開く
2. 日経225の価格が自動的に取得されます
3. 取引株数を入力して「買う」または「売る」ボタンをクリック
4. 総資産を増やすことを目指しましょう！

## 技術スタック

- HTML5
- CSS3（レスポンシブデザイン）
- Vanilla JavaScript（ES6+）
- Yahoo Finance API（価格データ取得）
- LocalStorage（データ永続化）

## データソース

このゲームは Yahoo Finance API から日経225（^N225）の価格データを取得しています。
APIが利用できない場合は、前回取得した価格を表示します。

### 価格データの処理

- Yahoo Finance APIは^N225の価格をUSDで返すことがあります
- アプリケーションは自動的に通貨を検出し、必要に応じてJPYに変換します
- USD/JPY為替レートは動的に取得され、最新のレートで変換されます
- 為替レート取得に失敗した場合は、概算レート（150円/USD）を使用します
- 価格データの妥当性チェック（日経225の現実的な範囲: 15,000〜60,000円）を実施します
- API接続失敗時は前回の価格を保持し、一貫性のある表示を維持します
- 初回起動時のみ、APIが利用できない場合は概算値（39,000円）を使用します
